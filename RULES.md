Sunday competition rules
========================

Goals:
* Try something of your own
* Be awesome to each other and future folks

Rules:
* Any size teams you want, but the prize is two tickets
* 40% for the idea
  * Do we think it's fun / interesting / cool / etc
  * Can do a demo to boost this if you want
* 40% for the write up
  * Maker style write up - git repo or webpage or both
  * Not an academic presentation
  * Negative marks for powerpoint unless you've got a really good reason
  * Goal is to make it so other people could make something similar
* 20% being helpful
  * Helping other people
  * PRs, in person help, being considerate, etc

Close time will be added here as soon as I know it but likely around 5pm
